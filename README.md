# Currency Value Converter 

How much money would you need in New York to buy the same things you'd buy in India? 

Usage:

```
$ pip install -r requirements.txt
$ ./website.py runserver
$ ./manage.py importcsv -f data.csv
$ ./manage.py runserver
```
